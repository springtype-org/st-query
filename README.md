<h1 align="center">SpringType: st-query</h1>

> DOM mutation API that feels like the good old jQuery

[![Gitter](https://badges.gitter.im/springtype-official/springtype.svg)](https://gitter.im/springtype-official/springtype?utm_source=badge&utm_medium=badge&utm_campaign=pr-badge)

<h2 align="center">Purpose</h2>

This micro-library is an extension for the SpringType v3 JSX/TSX microframework. This API is basically syntax-sugar for developers who prefer the good old jQuery-like programming style over the native DOM API. 

<h2 align="center">Features</h2>

- [x] Supports JSX/TSX for DOM `innerHTML` updates
- [x] Ultra-fast: Uses SpringType `ref` instead of DOM CSS selector queries
- [x] Supports standard jQuery API for the most usual tasks
- [x] Super tiny ~`725 bytes` (gzip) all-in (!)
- [x] Zero runtime NPM dependencies
- [x] First class TypeScript support
